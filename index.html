DrillDowner.js

A dependency-free ES6 widget for interactive, hierarchical drill-down tables and ledgers.

View on GitHub

Examples

  • Simple Example — Basic product inventory organized by department with totals and formatting.
  • Medium Example — Order management with regional customer grouping and switchable view modes.
  • Advanced Example — Financial ledger with running balance, multiple views, A-Z navigation, and click callbacks.
  • DrillDowner Example — Comprehensive inventory with multiple grouping hierarchies and A-Z navigation sidebar.
  • Books Example — Book inventory with multiple groupings, custom sorting, and formatted currency totals.
  • TogglesUp Example — Shows how unique child values bubble up and appear in parent group rows.
  • Label Click Callback — Captures clicks on group labels and shows detailed context in a modal dialog.
  • Balance Example — Bank statement with row-by-row running balance in ledger mode and net totals in group view.
  • Balance Example 2 — Income/expense statement with running balances that accumulate across grouped and ledger views.
  • Ledger Drill Example — Transaction ledger sorted latest-first with category analysis and running balance.
  • Example — Sales dashboard with region-to-product hierarchies, formatted currency, and status badges.
  • Parking Area Example — Drag dimensions in/out of grouping, check to show parked dims as extra columns, drag to reorder them.
